# 安装
## Windows
使用[BetterNCM Installer](https://github.com/MicroCBer/BetterNCM-Installer)一键安装~

安装完成后在网易云音乐-设置-最下面 安装插件

# 开发
请参考插件库中的插件

将你的插件/主题放置在对应位置即可

大型插件开发参考 https://github.com/MicroCBer/NeteaseLiveSongPlayer

想要添加主题/插件到主题商店的，可以联系我
